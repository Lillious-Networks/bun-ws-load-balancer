# bun-load-balancer

To install dependencies:

```bash
bun install
```

Build for windows:
```
bun build-windows
```

Build for linux:
```
bun build-linux
```

Build for Darwin:
```
bun build-darwin
```

Build for all operating systems:
```
bun build
```

More detailed build steps are located in the **package.json** file

Required Environment Variables:
```
KEY
ENCRYPTION_KEY
HOST
REALM_PORT
SHARD_PORT
```
